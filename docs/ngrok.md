# Показать сайт снаружи через ngrok

Быстрый способ дать доступ к локальному сайту из интернета по HTTPS-ссылке —
**без проброса портов на роутере** и без светящегося домашнего IP.

## Предусловие

Стек должен быть поднят и работать локально на порту 80:

```bash
cd stroy-website
docker compose up -d
# проверка: http://localhost/ открывается
```

## 1. Установить ngrok (один раз)

**Windows (winget):**
```bash
winget install ngrok.ngrok
```
Или скачать с https://ngrok.com/download и распаковать.

## 2. Привязать аккаунт (один раз)

ngrok с бесплатным тарифом требует токен:

1. Зарегистрироваться: https://dashboard.ngrok.com/signup
2. Скопировать свой токен: https://dashboard.ngrok.com/get-started/your-authtoken
3. Прописать его:
```bash
ngrok config add-authtoken <ТВОЙ_ТОКЕН>
```

## 3. Запустить туннель

```bash
ngrok http 80
```

ngrok выведет публичную ссылку вида:
```
Forwarding   https://abc123.ngrok-free.app -> http://localhost:80
```

**Эту `https://...ngrok-free.app` ссылку и кидаешь** кому хочешь показать.
Сайт работает: лендинг, портфолио, картинки (`/uploads`), API (`/api`) —
всё через тот же origin, mixed-content нет (ngrok отдаёт по HTTPS).

Терминал с ngrok **держи открытым** — закроешь, туннель отвалится.
Останавливать: `Ctrl+C`.

## Нюансы

- **Первый экран ngrok.** На бесплатном тарифе при первом заходе браузер
  показывает страницу-предупреждение ngrok с кнопкой «Visit Site» — это норма,
  гость жмёт кнопку и попадает на сайт.
- **Ссылка меняется** при каждом перезапуске `ngrok http 80` (на free-тарифе).
  Запустил → скинул свежую ссылку.
- **Безопасность.** Через туннель наружу торчит и **админка** (`/admin/login`).
  Трафик шифруется (HTTPS ngrok), но:
  - Смени демо-пароль `ADMIN_PASSWORD` в `.env` на свой и перезапусти
    `docker compose up -d` перед тем как давать ссылку чужим.
  - Не держи туннель поднятым 24/7 «просто так» — поднял, показал, выключил.
- **Порт.** Если занял 80 другим — поднимай стек на другом порту и туннелируй его
  (`ngrok http <порт>`).

## Альтернатива без регистрации

Если не хочешь заводить аккаунт ngrok — Cloudflare Tunnel разово:
```bash
cloudflared tunnel --url http://localhost:80
```
Выдаёт `https://<random>.trycloudflare.com` без логина (но ссылка тоже временная).
