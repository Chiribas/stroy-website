# Правила и заметки

Сюда складываю всё что нужно помнить при работе над проектом.

## Логи и анализ

- При ошибках в контейнерах: `docker compose logs -f <service>`
- Healthcheck: `curl http://localhost:8080/health`
- Логи nginx: `docker compose logs nginx`

## Локальная отладка

При локальной разработке используем порты:
- Frontend dev: `npm run dev` → http://localhost:3001
- Backend dev: `dotnet run` → http://localhost:8081

Docker Compose использует стандартные порты (80, 3000, 8080) — контейнеры изолированы.

## Правила добавления

- Всегда добавлять новые инструкции сюда как только вспоминаем
- Ничего не забывать!
